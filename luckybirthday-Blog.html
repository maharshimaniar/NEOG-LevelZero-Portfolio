Writing JavaScript

First, we will wire Inputs, Button and Output-Div to JavaScript with help of .querySelector() method. This method binds DOM (Document Object Model) elements with JavaScript. 


const dateOfBirth = document
                    .querySelector("#date-of-birth");
const luckyNumber = document
                    .querySelector("#lucky-number");
const checkButton = document
                    .querySelector("#calculate-number");
const displayMessage = document
                    .querySelector("#output-container");

Next, We obtain the Sum of all Digits of Date. Here we have used In-Line function. 


const calculateSum = (date) => {
    let sum = 0;
    date = date.replaceAll("-", "");
    for (let digit of date) {
        sum = sum + Number(digit);
    }
    return sum;
    };

In this, .replaceAll() method replaces all '-' in date string. Then the for loop typecastes each character into Number and adds it to sum variable. And at last this sum is reaturned.

Next, we will check if sum is divisible by Lucky number or not. Here also we will use Inline function. Here showMessage is a functon, which will take an Input message and sets it as Output-Div with help of .innerHTML() method.

Lastly, the button click mechanism is binded to Button with help of .addEventListner() method. If any input field is missing, then eror message will be shown with help of showMessage() function. Else, first sumOfDate() andcheckIsNumberLucky() functions will be invoked. 


checkButton.addEventListener("click", () => {
    const date = dateOfBirth.value;
    const numberToCheck = luckyNumber.value;
    if (date && numberToCheck) {
        const sumOfDate = calculateSum(date);
        checkIsNumberLucky(sumOfDate, numberToCheck);
    } else {
        showMessage("Please enter both the fields");
    }
});
